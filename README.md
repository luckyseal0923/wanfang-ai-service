# WorkHub｜AI 與自動化院內工作平台

WorkHub 是醫院內部多功能行政與臨床輔助系統的**入口整合平台（Portal）**。旨在整合各項已啟用的 AI 與自動化流程工具，並列出開發中與規劃中的系統，讓院內同仁能在此一站式存取所需的行政與會診工具。

---

## 🌐 平台資訊

* **平台創作者**：張智維（GSM=6524）
* **技術堆疊**：原生 HTML / CSS / JavaScript (採用 Lucide Icons 圖標庫)

## 🛠️ 開發與規劃中系統

平台預留了以下院內流程的自動化對接版塊：
1. **遠端文件簽核系統**：支援 PDF 上傳、設定簽名框與手機遠端簽名。
2. **EMS橋接平台**：院前救護端任務建立、病人資訊傳遞與院內即時回報。
3. **個案轉介平台**：醫療端起案、轉介資訊整理、機構媒合與進度追蹤。
4. **智慧護理交班系統**：病人重點處置、交班紀錄與待辦事項數位化整合。
5. **IV 團隊預約平台**：靜脈治療相關作業的預約、時段安排與任務分派。
6. **電子抽獎系統**：尾牙與院內活動抽獎名單匯入、電子摸彩與中獎紀錄。

2. **託管部署**：可直接將此儲存庫對接到 GitHub Pages 或 Zeabur 進行靜態網頁部署。
